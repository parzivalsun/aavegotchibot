EN
1) Install python 3.9 (https://www.python.org/downloads/release/python-3913/)
2) Install any IDE, I use pycharm (https://www.jetbrains.com/en-us/pycharm/download/) and it's better to run through pycharm, because this bot was written there.
3) Create a new project in the paycharm, then copy all the downloaded files to the folder with the newly created project, so that they will be loaded in your paycharm
4) unpack ustils.zip to the root of drive C
5) Install requirements.txt libraries
6) File accounts.txt fill in the format mnemonic_phase@proxy. Important, this bot does not work with proxies that require authentication, bind to your IP
7) Before starting the bot, you need to set the screen resolution to 4K (3840x2160), if your screen does not support this resolution, you need to create it in the nvidia or amd control panel (a detailed guide on this can be found in Google without problems)

RU
1) Установить python 3.9 (https://www.python.org/downloads/release/python-3913/)
2) Установить любую IDE, я использую pycharm (https://www.jetbrains.com/ru-ru/pycharm/download/) и лучше запускать через pycharm, потому что этот бот был написан там.
3) Создаете новый проект в пейчарм, после чего копируете все скачанные файлы в папку с только созданным проектом, чтобы в пейчарме они у вас загрузились
4) ustils.zip распаковать в корень диска C
5) Установить библиотеки requirements.txt
6) Файл accounts.txt заполнить в формате mnemonic_phase@proxy. Важно, данный бот не работает с прокси которым требуется аутенфикация, делайте привязку к вашему IP
7) Перед запуском бота необходимо установить разрешение экрана 4К (3840х2160), если ваш экран не поддерживает данное разрешение необходимо создать его в панели управления nvidia или amd(подробный гайд об этом можно найти в гугле без проблем)
